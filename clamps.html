    • Features

    1. User Authentication: Users must enter their password correctly to create and edit new entries.

    2. Account Restricting: Only one account is allowed per email.

    3. Theme: The website has a consistent black, white, red theme to represent security and to keep simplicity.

    4. Entry Editing: Users are able to edit their entries.

    5. Button Navigation: Users are able to navigate the website and its pages through the use of 'Homepage buttons' at every page.

    6. Light/Dark Mode: Users are able to view the website in a dark or light theme.

    7. Search: Users are able to search for their entries based on name.

    8. Confirmation: Users will be prompted a confirmation dialog when an entry has been successfully saved.

    9. Tab Navigation: Users can navigate the website through the use of tabs at the top of the page.

    10. Password Strengthener: Users are required to have a certain password criteria when creating entry passwords.

    11. Security Q&A: Users must provide a security question and answer when authenticating themselves.
